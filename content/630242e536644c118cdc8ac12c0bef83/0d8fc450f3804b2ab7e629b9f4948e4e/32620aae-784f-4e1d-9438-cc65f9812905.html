Listed below are various cloud attack tools:

Nimbostratus

Source: https://andresriancho.github.io

Nimbostratus is a tool used for fingerprinting and exploiting Amazon cloud infrastructures.

It allows attackers to

  • Enumerate access to AWS services for the current IAM role.
  • Use a poorly configured IAM role to create new AWS user.
  • Extract current AWS credentials from metadata, .boto.cfg files, environment variables, etc.
  • Clone DBs to access information stored in snapshot, etc.
